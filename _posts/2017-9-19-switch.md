**switch语句后面的表达式与case语句后面的表示式，在比较运行结果时，采用的是严格相等运算符（===），而不是相等运算符（==），这意味着比较时不会发生类型转换。**

    ```js
    var x = 1;

    switch (x) {
      case true:
        console.log('x发生类型转换');
      default:
        console.log('x没有发生类型转换');
    }
    // x没有发生类型转换
    ```
